# [Math](https://math-brtmvdl.surge.sh/)

## how to use

Install [Docker Compose](https://docs.docker.com/compose/install/)

```
git clone https://github.com/brtmvdl/math.git
cd math
bash env/up.sh
```

Open in `localhost:8080`
